
# Studi Kasus Basis Data: Warung Bakso Mang Ujang

Proyek ini merupakan implementasi sistem basis data untuk digitalisasi operasional sebuah usaha kuliner mikro (UMKM) bernama **Warung Bakso Mang Ujang** yang berlokasi di Garut. Studi kasus ini dilakukan sebagai pemenuhan tugas akhir mata kuliah **Basis Data** di **Institut Teknologi Garut**.

## Struktur Proyek

### 1. **Dokumentasi Proyek**
- `LAPORAN UAS BASIS DATA KELOMPOK 5.pdf`  
  Berisi seluruh isi laporan studi kasus: dari latar belakang, teori, ERD, normalisasi, implementasi, hingga kesimpulan & saran.

### 2. **Desain dan Perancangan**
- `ERD`  
  Entity Relationship Diagram (ERD) mencakup 8 entitas: Produk, Produksi, Bahan Baku, Supplier, Pengiriman, Keuangan, Karyawan, Divisi.
- `Normalisasi.xlsx`  
  File Excel yang menunjukkan proses normalisasi dari UNF → 1NF → 2NF → 3NF.

### 3. **Implementasi HTML (Prototype Antarmuka)**
- `produk.html` – Form input data produk.
- `produksi.html` – Form input produksi harian.
- `karyawan.html` – Form input data karyawan.
- `keuangan.html` – Form input data keuangan dan gaji.

### 4. **Transkrip Wawancara**
- Terdapat dalam laporan PDF (Halaman Lampiran).
- Berisi hasil wawancara dengan pemilik usaha untuk mengidentifikasi kebutuhan sistem.

---

## Teknologi & Tools
- **DBMS**: MySQL (via phpMyAdmin)
- **Desain ERD**: Draw.io
- **Normalisasi**: Microsoft Excel
- **Editor SQL**: Visual Studio Code
- **Prototype**: HTML Form sederhana

---

## Fitur yang Diimplementasikan
- Struktur database relasional 8 entitas.
- Query analitik:
  - Total produksi per produk
  - Total gaji per jenis pekerjaan
  - Jumlah karyawan per divisi
- Insert, Update, Delete data
- Antarmuka input data (produk, produksi, karyawan, keuangan)
